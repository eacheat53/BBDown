# 社区行为准则

本项目致力于提供一个友好、包容、无骚扰的协作环境。

## 我们的承诺

作为项目维护者和贡献者，我们承诺：让参与本项目的每个人都获得无骚扰的体验，无论其年龄、体型、残疾、种族、性别认同与表达、经验水平、国籍、个人外貌、种族、宗教或性取向如何。

## 不可接受的行为

- 使用涉及性取向的言语或图像
- 挑衅、侮辱、贬损性评论，以及人身攻击
- 公开或私下骚扰
- 发布他人的私人信息（如住址、邮箱）
- 其他在专业环境中不合理的行为

## 执行

项目维护者有权利和义务删除、编辑或拒绝不符合本准则的评论、提交、代码、Issue 和其他贡献，并暂时或永久地禁止贡献者参与本项目。

本行为准则适用于所有项目空间，也适用于个人在公共场所代表项目时的行为。

## 适用范围

本准则适用于项目仓库、讨论区、PR 审查、Issue 区以及任何与项目相关的公开或私下交流。

## 联系方式

如有违规事件，请通过项目维护者 GitHub 账号的公开渠道联系。
